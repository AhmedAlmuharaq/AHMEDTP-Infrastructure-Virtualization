## Labs Included

### TP1 — Kubernetes Cluster Setup

This lab focused on building a Kubernetes cluster from scratch.

Main tasks:

- preparing Linux machines  
- configuring kernel modules and sysctl  
- installing container runtime  
- installing kubeadm, kubelet, kubectl  
- initializing the control plane  
- joining worker nodes  
- installing network plugin  
- validating cluster status  

---

### TP2 — Kubernetes Application Deployment

This lab focused on deploying and running an application inside the cluster.

Main tasks:

- preparing application files (HTML / PHP / SQL)  
- creating Kubernetes manifests  
- deploying pods and services  
- exposing the application  
- testing access and functionality  
- capturing screenshots for validation  

---

### TP3 — Nextcloud / PostgreSQL / Persistence

This lab focused on deploying a real-world service with persistence.

Main tasks:

- preparing PostgreSQL database  
- deploying Nextcloud  
- linking services together  
- managing persistent storage  
- validating data persistence  
- documenting results with screenshots
